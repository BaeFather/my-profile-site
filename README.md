# Jayson.B — 개인 프로필 사이트

풀스택 개발자 Jayson.B의 개인 포트폴리오 & 프로필 웹사이트입니다.  
빌드 도구 없는 순수 정적 사이트(HTML/CSS/JS)로 구성되어 있습니다.

---

## 미리보기

| 섹션 | 설명 |
|------|------|
| Hero | 타이핑 애니메이션 + 프로필 사진 |
| About | 간략한 자기소개 및 키워드 뱃지 |
| Skills | 핵심 역량 / 부가 역량 기술 스택 목록 |
| Portfolio | 주요 프로젝트 카드 (커머스, 인프라, 대시보드) |
| Contact | 이메일 / GitHub 연락처 링크 |

---

## 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| 마크업 | HTML5 |
| 스타일 | Tailwind CSS (CDN), 커스텀 CSS |
| 스크립트 | Vanilla JavaScript |
| 애니메이션 | AOS (Animate on Scroll) 2.3.1 |
| 폰트 | Google Fonts — Inter |

> **인터넷 연결 필수** — CDN 의존성으로 인해 오프라인 환경에서는 스타일/애니메이션이 적용되지 않습니다.

---

## 파일 구조

```
my-profile-site/
├── index.html    # 전체 페이지 구조 및 섹션
├── style.css     # 커스텀 CSS (타이핑 커서, 호버 효과 등 Tailwind 불가 항목)
├── script.js     # 타이핑 애니메이션, 네비게이션 스크롤, AOS 초기화
└── profile.jpg   # 프로필 사진 (Hero 섹션 표시)
```

## 실행 방법

빌드 과정이 없습니다. 아래 방법 중 하나를 사용하세요.

**방법 1 — 브라우저에서 직접 열기**

```
index.html 파일을 브라우저로 드래그 앤 드롭
```

**방법 2 — Python 로컬 서버**

```bash
python -m http.server 8080
```

이후 브라우저에서 `http://localhost:8080` 접속
배포 사이트 : `https://baefather.github.io/my-profile-site/`


**방법 3 — VS Code Live Server**

VS Code에서 `index.html`을 열고 우측 하단 **Go Live** 클릭

## 주요 기능

- **타이핑 애니메이션** — `풀스택 개발자 → 기술 리더 → 인프라 엔지니어` 순환
- **스크롤 감지 네비게이션** — 스크롤 위치에 따라 활성 메뉴 자동 하이라이트
- **모바일 반응형** — 햄버거 메뉴 및 모바일 레이아웃 지원
- **스크롤 진입 애니메이션** — AOS 라이브러리 기반 섹션별 페이드 효과
